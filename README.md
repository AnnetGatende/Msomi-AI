## Problem
Kenya has a teacher-to-student ratio of 1:58 in public schools.
Students lack personalized STEM support, especially in Coast Region.
Existing EdTech solutions use Western examples that don't connect
with Kenyan students.

## Solution
Msomi is an AI-powered STEM tutor for Kenyan secondary school
students (Form 1–4) that teaches using natural Swahili/Sheng
code-switching and localized Kenyan analogies.

## Features
- Maths, Physics, Biology, Chemistry (Form 1–4)
- Natural Swahili + Sheng + English responses
- Kenyan analogies: matatus, M-Pesa, ugali, boda bodas
- Subject locking — stays focused per subject
- Personalized by student name and Form level
- Session persistence — remembers you on refresh
- Mobile responsive design
